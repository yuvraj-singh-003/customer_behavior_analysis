# Customer Behavior Analysis

This repository presents a comprehensive analysis of customer behavior and related business questions using SQL, Python, and reporting tools.

---

### Project Highlights

- Provides an end-to-end analytics workflow: from raw data processing and database queries to advanced data exploration, visualization, and business reporting.
- Demonstrates practical use of SQL for answering real-world business questions relevant to customer engagement and behavior.
- Applies Python and Jupyter Notebooks for in-depth EDA (exploratory data analysis), feature engineering.
- Data visualization is delivered through interactive, user-friendly dashboards built in Power BI.
- Summarizes all key findings, insights, and business recommendations in a professionally formatted PDF for decision-makers.
- Designed for self-study, academic, or business audiences interested in data-driven customer analytics and reporting.

---

- ## Project Files
  
- [customer_behavior_analysis.ipynb](./customer_behavior_analysis.ipynb)  
  Jupyter Notebook covering data cleaning, tranformation, and feature engineering.

- [business_analysis.sql](./business_analysis.sql)  
  Contains SQL queries and scripts for exploring business data and generating key insights.

- [customer_behavior_report.pdf](./customer_behavior_report.pdf)  
  Final PDF report summarizing the findings, insights, and actionable recommendations from the analysis.
